Mobile PWA preview

ภาษาภาพเดียวกับ desktop แต่ปรับ thumb zone, bottom nav และ safe-area mindset

Thumb-first structure

Bottom nav มี 5 จุดหลัก, primary action อยู่ใกล้ช่วงล่าง และ screen แรกให้เห็น equity, P&L และ sync status ทันที

Warm dark surfaces

Mobile ใช้ surface และ border ชุดเดียวกับ desktop เพื่อไม่ให้ `/welcome`, `/portfolio` และ `/settings` ดูเป็นคนละแอป

Restrained accent

ทองถูกใช้กับ active tab, CTA และ state สำคัญเท่านั้น ไม่ใช้เป็น glow หรือ decoration กระจายทั้งจอ

Design system notes

token, component และ exception ที่ควรถูกย้ายกลับไป `src/app.css` และ Tailwind

Core palette

พื้นหลังและ surface เป็น warm black/charcoal ไม่ใช่ blue slate

Accent discipline

ทองใช้กับ active state, CTA และ signal สำคัญ ไม่ใช้กับทุก heading

Semantic colors

เขียว/เหลือง/แดงใช้เฉพาะ status และ risk/action ไม่ใช่สีประดับ
